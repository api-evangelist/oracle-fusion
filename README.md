# Oracle Fusion Cloud Applications (oracle-fusion)
Oracle Fusion Cloud Applications represent a comprehensive suite of cloud-based enterprise resource planning (ERP), human capital management (HCM), customer experience (CX), and supply chain management (SCM) solutions.

**URL:** [Visit APIs.json URL](https://www.oracle.com/apis/fusion)
